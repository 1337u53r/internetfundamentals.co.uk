Web Accessibility Initiative (WAI)

W3C which stands for World Wide Web Consortium is a web standards organization founded by Sir Tim Berners Lee, after he created the World Wide Web. The W3C organisation currently consists of 483 international members ranging from universities to charities and even commercial organisations. The Web Accessibility Initiative was created as a way of standardising how websites should be designed so whatever language, software, hardware, location or ability a user has they all get the same experience. By achieving this, developers can be certain that their website is accessible to anyone with any sort of disability such blindness, lack of sight, movement or cognitive ability.
