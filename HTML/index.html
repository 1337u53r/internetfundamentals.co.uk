What is HTML?

HTML which stands for HyperText Markup Language is a markup language used for front-end web development (web pages and applications). There have been many iterations of HTML over the years however HTML5 and HTML4 are the most recent ones, HTML5 being the newest. HTML5 was introduced in 2014 to replace HTML4, it includes many improvements and features such as updated/new elements (tags), simplified syntax etc.
